Política de Privacidad de Divicuenta

Última actualización: Noviembre 2025

Divicuenta es una aplicación diseñada para dividir cuentas de manera simple y rápida entre varias personas. Esta política describe cómo se maneja la información dentro de la app.

1. Información que recopilamos

La aplicación NO recopila ni almacena información personal identificable. Toda la información ingresada por el usuario (como montos, nombres, items consumidos) se procesa únicamente dentro del dispositivo del usuario.

2. Almacenamiento de datos

Todos los datos utilizados por la aplicación se mantienen de forma local en el dispositivo. Divicuenta no utiliza servidores propios ni bases de datos externas.

3. Permisos utilizados

Divicuenta utiliza únicamente los siguientes permisos:

  • Internet: requerido por los anuncios (AdMob).
  • Acceso a red: para mostrar publicidad.

No se solicitan permisos de cámara ni ubicación.

4. Publicidad

La aplicación utiliza Google AdMob para mostrar anuncios (banner e intersticiales). Google puede recopilar información anónima para personalizar la publicidad. Más información sobre cómo Google maneja datos:

https://policies.google.com/technologies/ads

5. Terceros

No compartimos información personal con terceros, ya que no recopilamos información personal.

6. Seguridad

Dado que toda la información se procesa y almacena localmente, la seguridad depende directamente del dispositivo del usuario.

7. Cambios en esta política

Podemos actualizar esta política ocasionalmente. La fecha de actualización aparecerá siempre al inicio de este documento.
